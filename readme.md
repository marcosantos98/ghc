# ghc - Github clone

Little command line tool that clones repos only with user/repo

## Usage:

### Build:

```console
cmake .
make
```

### Run:

```console
./ghc -r mscode98/ghc
```

#### Note: 
- Use `-hs` to use HTTPS instead of SSH.
- Run `sudo make install` to install ghc system wide.

### Software use:

- Argparse by [@p-ranav](https://github.com/p-ranav)
